# 「Vibe Coding 主題式廣徵提案」計畫書

## 一、 提案主題
**【Vibe Coding 主題式廣徵提案】北捷品牌數位管家：CI 規範查詢與介面原型生成平台**

## 二、 提案背景與問題說明
*   **品牌維護困難**：同仁在製作內部系統、報表或宣傳網頁時，難以精確掌握公司 CI（企業識別）規範（如色碼、字體、Logo 使用限制）。
*   **跨單位溝通成本高**：行政或業務單位與開發單位溝通介面需求時，常因缺乏視覺原型導致認知偏差，反覆修改耗費時日。
*   **數位轉型門檻**：非資訊背景同仁想要開發簡易資訊看板或資料庫時，往往受限於介面設計（UI/UX）能力，導致產出工具專業感不足。

## 三、 問題分析與解決方案
本提案透過 Vibe Coding 技術（AI 輔助開發），打造一個「北捷品牌資源管家」數位工具，解決方案包含：

1.  **品牌卡自動化管理**：整合所有官方色號（主視覺藍、捷運藍、永續綠等），提供一鍵複製色碼與 HSV 數值分析，確保色彩應用精確。
2.  **品牌組合預覽模擬器**：可即時切換不同 Logo 形式與字體組合，讓同仁在開發前即可預覽品牌視覺效果，降低設計錯誤率。
3.  **一鍵生成品牌系統版型**：
    *   內建針對 **行控中心 (OCC)**、**車站服務處 (Station)**、**維修排程 (Maintenance)** 等不同業務場景的介面版型。
    *   使用者只需選擇需求，系統即自動生成符合 CI 規範的 HTML/CSS 原始碼，非資訊同仁也能快速搭建專業的數位工具原型。

### [AI 製作過程說明]
本工具開發過程完全採用自然語言描述北捷 CI 規範，由 AI 輔助完成複雜的色彩分類邏輯（如 HSV 排序算法）與動態 SVG 圖表產生器，並利用 AI 快速生成多種場景的網頁佈局程式碼。

## 四、 效益分析
1.  **提升行政效率**：將原本需查詢 PDF 規範的過程縮短為秒級的點擊操作，預計可節省 90% 的規範核對時間。
2.  **確保品牌一致性**：從根源解決各單位數位工具「品牌顏色五花八門」的問題，維護台北捷運專業、一致的企業形象。
3.  **賦能非技術人員**：同仁不需具備專業美編或前端開發知識，即可產出具有專業系統感（Dashboard）的工具，加速部門內的數位轉型。
